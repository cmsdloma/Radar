## 📱 About the App

**Radar** is a peer-to-peer mesh communication and location-sharing Android app that works
over Bluetooth LE, Wi-Fi Aware, local network, and internet relay. It provides messaging,
emergency/SOS tools, a live peer map, and optional Matrix chat integration.

- **Package ID:** `uk.co.daltechnology.radar`
- **Developer:** DAL Technology
- **Platform:** Android

---

## 🔑 Key Privacy Principles

- Peer identity is **pseudonymous** — only a randomly generated UUID, optional nickname, and
  optional gender are stored on the server. No email address or phone number is required or stored.
- Your **current location** is stored on the server to enable peer discovery (current value only —
  no location history is retained).
- **Messages relayed via the server are auto-deleted within 7 days**, or sooner once all recipients
  acknowledge receipt.
- **End-to-end encrypted (sealed-sender) messages** are stored as opaque ciphertext — the server
  cannot read the content or determine the real-world identity of participants.
- **SMS content is never uploaded** to our servers.
- We do **not** sell data, use it for advertising, or share it with data brokers.
